Requirements

- Python 3.x
- 'requests' library
- 'pandas' library
- openpyxl library
- git

You can install the required library using:
pip install requests pandas openpyxl

Installation

    1. Create a working directory to work in
    mkdir <folder name>

    2. Clone the repository:
    git clone https://github.com/woobrandon/GovTechAssessmentBrandon.git

    3. Check that both app.py and Country-Code.xlsx file is in the folder

How to use:
    1. Run the python script
        python app.py

    2. 2 new csv files (restaurant_events.csv, restaurant.csv) will be created in the current directory

    3. An output will display in the terminal showing the threshold for the 5 rating groups
